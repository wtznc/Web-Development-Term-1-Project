What aspect(s) of your work do you think worked well and why?

Smooth JavaScript and CSS animations - improve overall user experience.

Responsive web design - every element of the website scales according to the viewport size.

Clarity and minimal design - it is crystal clear about what exactly you can do on my webpage.

What aspect(s) of your work could be improved, and how might you do things differently another time?

I could have used Handlebars and create templates for the grid with future projects. The final version of this website will store data in database.

Next time, I'd like to load images and divs dynamically while scrolling. I think I will need only basic knowledge about JavaScript and jQuery.
